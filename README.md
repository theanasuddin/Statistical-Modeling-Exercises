
# Statistical Modeling  
### Exercise Solutions & R Implementations

This repository contains my solutions for the **[Statistical Modeling](https://opiskelijanopas.tuni.fi/en/tampere-university/curriculum/course-units/uta-ykoodi-48007?year=2025)** course.  
The work focuses on applying statistical modeling techniques for regression, classification, count data, categorical outcomes, and survival analysis using R.

---

## Topics Covered

- Linear regression models  
- Nonlinear normal models  
- Generalized linear models (GLMs)  
- Gamma and other positive‑response models  
- Poisson and zero‑inflated count models  
- Logistic regression  
- Survival data analysis  

---

## Repository Structure

Each folder includes:
- R markdown code in runnable format  
- Model outputs, plots, and diagnostics  
- Written explanations and analysis

## Technologies Used

- **RStudio**
